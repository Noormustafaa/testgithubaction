step 13.
inside a src folder Create the __init__.py file and math_operations.py file 
and inside math_operations.py write 
def add(a, b):
    return a + b


def sub(a, b):
    return a - b



step14.

create file in test folder __init.py and test_operations.py file
in test folder write this
from src.math_operations import add,sub

def test_add():
    assert add(2, 3) == 5
    assert add(-1, 1) == 0


def test_sub():
    assert sub(5, 3) == 2
    assert sub(4, 3) == 1
    assert sub(3, 3) == 0
    assert sub(2, 3) == -1
